## 🚀 Featured Projects

### 📱 1. Human Activity Recognition System
**Data Science & Machine Learning Project**

A smartphone-based classification system that processes acceleration and angular velocity signals from 30 subjects to accurately recognize 6 distinct human activities.

* **The Data:** 561 highly normalized, numeric features extracted from smartphone sensors (Body acceleration, Gyroscope, Jerk signals, FFT transforms).
* **Methodology & Models:**
    * Applied **Principal Component Analysis (PCA)** for feature dimensionality reduction.
    * **Logistic Regression:** Built a linear classifier reaching **90% accuracy**, serving as an excellent baseline but occasionally confusing similar tracking data (like sitting vs. standing).
    * **Neural Network:** Developed a non-linear network featuring 2 hidden layers (64 & 32 units), ReLU activation, Dropout regularizations, and an Adam optimizer. 
* **Result:** The Neural Network successfully learned complex patterns, achieving a powerful **97% validation accuracy** and proving to be the optimal fit for high-dimensional sensor data.

### 🎓 2. EventDhundo — Campus Discovery Platform
**Full-Stack Software Engineering Project**

A centralized campus event discovery and automated achievement portfolio platform designed to eliminate fragmented communication channels for university students and organizers.

* **My Role:** Frontend Developer & UI/UX Designer
* **The Architecture:** Built using **React.js** with **Material-UI/Tailwind CSS** for a completely responsive single-page experience, backed by a Node.js/Express.js RESTful API and a relational SQL database.
* **Key Features Implemented:**
    * **Personalized Recommendation Engine:** Displays a custom "Recommended for You" section based on user department, chosen interests, and historical attendance data.
    * **QR-Code Attendance Tracking:** Generates distinct QR codes for verified event entry, linking seamlessly to a database that automates student certificate generation.
    * **Digital Achievement Portfolio:** Automatically aggregates user event data into a beautiful, public-facing timeline exportable as a professional PDF resume.
